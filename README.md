# End-to-End Retail Data Analysis

## Overview

This project presents a comprehensive analysis of retail business data, covering sales, customers, products, stores, and returns. It demonstrates the complete data analytics workflow, including data preparation, modeling, querying, and insight generation to support data-driven decision-making.

---

## Objectives

* Evaluate overall sales performance
* Identify patterns in product returns
* Analyze customer behavior and demographics
* Assess store-level performance
* Generate actionable business insights

## Data Model

The project is designed using a **Star Schema**:

* **Fact Table:** sales_cleaned
* **Dimension Tables:** customers_cleaned, products_cleaned, stores_cleaned, returns_cleaned1

The ER diagram included in the repository illustrates relationships between all tables.
